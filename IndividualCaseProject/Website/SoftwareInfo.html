Visual Studio 2019 and VS Code

We utilize Microsofts Visual Stuido 2019 paired with Unity. Visual Studio has an actual Default package you download at install for coding games. This package is geared towards Unity and once you download Unity you get another package so that when you go to edit "scripts", it opens VS for you. This is definetly the IDE to use when coding C#. In our opinion you should not utilize any others.

Blender 2.93LTS

Blender is a free Open Source software that is used to create Game assets and Character models. Blender works well with Unity5 and provides a great enviornment to create almost anything. It has every feature you would need to create with. From your general modeling tools, to sculpting, to modifiers to visuallize changes without actually making them. It also is a full Renderer and Animation Software, Which means you don't need to learn and utilize another software to animate your stuff. It cuts down the learning curve which also increases efficiency.
